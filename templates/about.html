Web Application Technologies

This web application was built using the Python and Javascript programming languages. It uses the spaCy python library for natural language processing in order to parse learning outcomes and identify verbs. Notably, alongside verbs which appear in infinitive form, this allows for detection of any conjugated verb in a learning outcome. Since the underlying models of the spaCy library are statistical, the analyses delivered by this web application will also be statistical.

The back-end of the application was built using the Django framework, and Celery, for asynchronous task management.

The front-end of the application was built using HTML 5, CSS 3 and 'vanilla' Javascript, using AJAX for asynchronous processing. The analysis charts are generated using the Chartjs and D3js Javascript libraries.

Site Terminology

Below are a few terms you may encounter while you are on this site:

Learning Outcome
A statement of what the learner is expected to know, understand or be able to do on successful completion of a course or programme.
Taxonomy
A hierarchical model used to classify educational learning outcomes into levels of complexity and specificity.
Taxonomy Level
An integer denoting the hierarchical level of abstraction of the verb category within the taxonomy.
Taxonomy Overlap
The total number of unique instances across the taxonomy where a verb appears simultaneously in 2 categories. For instance, a verb appearing in exactly 2 categories contributes a factor of 1 to the total overlap. However, a verb appearing in exactly 3 categories contributes a factor of 3.
